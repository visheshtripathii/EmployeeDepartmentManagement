This is an Employee Department Management module CRUD API

This project is made with the help of Spring Boot

These are the concepts which were used while making this project

1) Tomcat server (Inbuilt))

2) Spring MVC and Spring Web (Thymleaf)

3) Database : MySQL

4) Spring data jpa for storing data in database

5) Validation concept used for wrong input in registration form

6) Mapping of tables is done with the help of @OneToMany annotation

7) Exception handling done in every method related to fetching of data
